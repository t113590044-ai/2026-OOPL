# Game Introduction

玩家在保護地球免受外星人浪潮襲擊的任務中控制了加拉西普戰鬥機。
遊戲包括摧毀每個外星人的陣型，外星人俯衝向玩家，試圖擊中他們。

[遊戲連結](https://youtu.be/n9Z6dIL8OIw)
